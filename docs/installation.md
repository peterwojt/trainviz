# Installation

This page explains how to install **Trainlytics**, either for testing, development, or production use.

---

## Prerequisites

- **Python 3.9 or higher**  
- `pip` package manager  
- Optional: **virtual environment** recommended for isolation  

---

## Install from PyPI (Official Release)

Once Trainlytics is released on PyPI, you can install it directly:

```bash
pip install trainviz
```
